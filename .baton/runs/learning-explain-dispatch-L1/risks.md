## 비목표 (재확인)
L2 학습 체크포인트(멈춤/질문/수정), L3 스트리밍·Swift 학습 뷰, 연습문제/퀴즈, 설명 파일 별도 산출.

## 후속 (학습 로드맵)
- **L2**: 지정 체크포인트에서 멈춤 → 설명 읽고 이해/질문/수정 → 진행.
- **L3**: 실시간 스트리밍 + Swift 학습 뷰(추론 과정 라이브, 체크포인트 UI). 이전 stream-output 설계 재활용.
- 후속: diff 해설, 연습문제/퀴즈, (선택)튜터 요소.
